# Litarvan's LightDM Webkit2 theme

 **=> Screenshots below**

# Features

 - Arch logo splash
 - English/French language
 - Hour and date
 - User/Desktop switching (saved)
 - Shutdown/Reboot buttons
 - Fade in on display, fade out on session start
